# 📌 Responsive Card UI - React

A simple and responsive Card UI built using React and Tailwind CSS.  
The project demonstrates reusable components, props usage, and dynamic data rendering using JSON.

---

## 🚀 Features

- 🎨 Clean and modern card design
- 📱 Fully responsive layout
- ♻ Reusable Card component
- 🔄 Dynamic data rendering using JSON
- 🧩 Props used for passing data
- ⚡ Built with Vite for fast development

---

## 🛠 Tech Stack

- React
- Tailwind CSS
- JavaScript (ES6+)
- Vite
